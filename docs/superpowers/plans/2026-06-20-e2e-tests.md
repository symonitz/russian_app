# E2E Functional Test Suite Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Automated Playwright E2E tests that drive the real static app through each learning-mode journey, plus dataset-integrity guards and CI.

**Architecture:** Playwright (headless Chromium) loads the static `site/` via a `python3 -m http.server` webServer. Tests seed `localStorage` before navigation for determinism and derive expected values from `site/data/*.json` so content changes don't break them. A pytest file asserts invariants over the built data. CI runs node + the data tests + Playwright on push/PR.

**Tech Stack:** `@playwright/test` (ESM JS, no TypeScript), Python `pytest`, GitHub Actions.

## Global Constraints

- The repo's `package.json` has `"type": "module"` — all new JS (config, helpers, specs) is **ESM** (`import`/`export`, `export default`).
- E2E specs live in `tests/e2e/*.spec.js`; they import helpers from `./helpers.js` and `@playwright/test`.
- **Determinism:** seed `localStorage["ruslearn.v2"]` via `page.addInitScript` BEFORE `page.goto`; derive expectations from `site/data/*.json`; find tiles/words by text, never by position (the app shuffles).
- **Progress shape:** `{ vocab, letters, patterns, counter }`; a "known" card is `{ due: 100000, reps: 3, state: "known" }`.
- **Reading gate:** unlocks only when `Object.keys(P.vocab).length >= 3` (seed ≥3 vocab).
- **Audio is `new Audio()` (not in the DOM)** — never assert an `<audio>` element; assert the **network request** to `/audio/*.mp3` fires (proves the audio key resolved).
- **Combining acute** is U+0301 (`"́"`); ё is U+0451.
- Run E2E with `npx playwright test`; run data tests with `.venv/bin/python -m pytest`.
- Key selectors (verbatim from `site/app.js`): home tiles `[data-go="alphabet|reviews|listen|reading|patterns"]`, home badge `#b-alpha` (contains `/ 33` once data loads). Reviews: `#ans`,`#check`,`#verdict`,`#next`,`#iwr`, word in `.qcard .big`. Reading: `.passage`,`.rtoken`,`.rtoken.rnew`,`#wordpop`,`#read-next`, gate text "Learn a few words". Listen: `#lans`,`#lcheck`,`#lverdict`,`#li-speak`. Patterns: `.prompt-en`,`.ptiles .ptile`,`.pans`,`#pcheck`,`#pverdict`,`#pnext`. Alphabet: `.qcard .big`,`#a-reveal`,`.answer .ipa`,`#a-got`.

```javascript
import { readFileSync } from "node:fs";
import { fileURLToPath } from "node:url";
import { dirname, join } from "node:path";

const DATA = join(dirname(fileURLToPath(import.meta.url)), "..", "..", "site", "data");

export function loadData() {
  const read = (n) => JSON.parse(readFileSync(join(DATA, n), "utf8"));
  return {
    words: read("words.json"),
    reading: read("reading.json"),
    patterns: read("patterns.json"),
    alphabet: read("alphabet.json"),
  };
}

export function knownVocab(ids) {
  const v = {};
  for (const id of ids) v[id] = { due: 100000, reps: 3, state: "known" };
  return v;
}

export async function seedProgress(page, state) {
  const full = { vocab: {}, letters: {}, patterns: {}, counter: 0, ...state };
  await page.addInitScript((s) => localStorage.setItem("ruslearn.v2", JSON.stringify(s)), full);
}

// Navigate + wait until the dataset has loaded (home badge shows "/ 33").
export async function gotoApp(page) {
  await page.goto("/");
  await page.waitForFunction(() => /33/.test(document.querySelector("#b-alpha")?.textContent || ""));
}

// The first English alternate of a gloss (the app accepts comma/slash alternates).
export function firstGloss(gloss) {
  return gloss.split(/[,/;]| or /)[0].trim();
}
```

### Task 3: Reading journey (gate + stress marks + tap-to-gloss)

**Files:**

- [ ] **Step 1: Write the spec**

`tests/e2e/reading.spec.js`:
```javascript
import { test, expect } from "@playwright/test";
import { gotoApp, seedProgress, knownVocab } from "./helpers.js";

test("reading is gated until a few words are learned", async ({ page }) => {
  await gotoApp(page);
  await page.locator('[data-go="reading"]').click();
  await expect(page.locator("#reading-stage")).toContainText("Learn a few words");
});

test("passage shows stress marks and tap-to-gloss resolves", async ({ page }) => {
  await seedProgress(page, { vocab: knownVocab([1, 2, 3, 4, 5, 6, 7, 8]) });
  await gotoApp(page);
  await page.locator('[data-go="reading"]').click();
  await expect(page.locator(".passage")).toBeVisible();

  // Stress marks render in the passage (the accents feature).
  const passage = await page.locator(".passage").textContent();
  expect(passage).toMatch(/́/);

  // Tapping a word resolves its gloss despite the accent marks (guards the
  // stripAccent glossary lookup). Some function words / homographs have no
  // gloss, so accept any token that resolves — if the lookup were broken,
  // NONE would.
  const tokens = page.locator(".rtoken");
  const count = await tokens.count();
  let glossed = false;
  for (let i = 0; i < count; i++) {
    await tokens.nth(i).click();
    if (((await page.locator("#wordpop").textContent()) || "").includes("—")) {
      glossed = true;
      break;
    }
  }
  expect(glossed, "no tapped word resolved a gloss").toBeTruthy();

  // Advancing loads another passage.
  await page.locator("#read-next").click();
  await expect(page.locator(".passage")).toBeVisible();
});
```

```javascript
import { test, expect } from "@playwright/test";
import { gotoApp, loadData } from "./helpers.js";

const { patterns } = loadData();

test("building the correct sentence reveals the spoken form", async ({ page }) => {
  await gotoApp(page);
  await page.locator('[data-go="patterns"]').click();
  await expect(page.locator(".prompt-en")).toBeVisible();

  // The first un-met pattern (id 0) is shown; which item is random, so read the
  // prompt and look up its answer in the data.
  const prompt = (await page.locator(".prompt-en").textContent()).trim();
  const item = patterns[0].items.find((it) => it.prompt.trim() === prompt);
  expect(item, `no pattern item matches prompt "${prompt}"`).toBeTruthy();

  // Tap the answer tiles in order (found by exact text in the bank).
  for (const word of item.answer) {
    await page.locator(".ptiles .ptile", { hasText: new RegExp(`^${word}$`) }).first().click();
  }
  await page.locator("#pcheck").click();

  await expect(page.locator("#pverdict")).toHaveClass(/good/);
  await expect(page.locator("#pverdict")).toContainText(item.say);
  await expect(page.locator("#pnext")).toBeVisible();
});
```

### Task 7: Dataset-integrity guards (pytest)

**Files:**
- Create: `tests/test_dataset_integrity.py`

**Interfaces:**
- Consumes: nothing (reads `site/data/*.json` directly).

- [ ] **Step 1: Write the tests**

`tests/test_dataset_integrity.py`:
```python
"""Regression guards over the built dataset (site/data/*.json)."""
import json
from pathlib import Path

DATA = Path(__file__).resolve().parents[1] / "site" / "data"
ACUTE = "́"  # U+0301


def _load(name):
    return json.loads((DATA / name).read_text(encoding="utf-8"))


def test_no_audio_key_has_combining_acute():
    audio = _load("audio.json")
    bad = [k for k in audio if ACUTE in k]
    assert bad == [], f"audio keys must be accent-stripped, found: {bad[:5]}"


def test_audio_manifest_is_populated():
    assert len(_load("audio.json")) > 500


def test_yo_was_restored_in_vocab():
    words = _load("words.json")
    assert sum(1 for w in words if "ё" in w["stressed"]) >= 10


def test_stress_marks_present_in_vocab():
    words = _load("words.json")
    assert sum(1 for w in words if ACUTE in w["stressed"]) >= 100


def test_every_pattern_answer_is_nonempty_strings():
    for p in _load("patterns.json"):
        for item in p["items"]:
            ans = item["answer"]
            assert ans and all(isinstance(x, str) and x for x in ans), item


def test_reading_glossary_keys_are_accent_free():
    for e in _load("reading.json"):
        for k in (e.get("glossary") or {}):
            assert ACUTE not in k, f"glossary key has acute: {k!r}"
```

## Self-review notes
- **Spec coverage:** harness+determinism (T1); the 5 journeys — reviews (T2), reading incl. stress+gloss guard (T3), listen audio-resolution (T4), patterns (T5), alphabet (T6); dataset-integrity guards (T7); CI scoped to deployed product (T8). All spec sections covered.
- **Determinism:** seeds set before `gotoApp`; expectations derived from `loadData()`; tiles found by text. Listen seeds a single word for a deterministic answer; patterns reads the prompt to find the right answer.
- **Audio:** asserted via the `/audio/*.mp3` network request (not a DOM element), per the constraint.
- **Meaningful (not tautological):** each spec asserts an observable outcome that fails if the app regresses (verdict classes, gloss popup content, audio request, badge increment, stress mark present).
- **No `npm test` change:** worker `npm test` stays `node --test`; E2E runs via `npm run test:e2e` / `npx playwright test` so the two suites stay separate.
